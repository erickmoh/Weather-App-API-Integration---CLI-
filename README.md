# To-Do List Manager (CLI)

## Overview
This project is a command-line based To-Do List application built with Python. It allows users to manage tasks efficiently while demonstrating core programming concepts such as data persistence and user interaction.

## Features
- Add new tasks
- View all tasks with completion status
- Mark tasks as completed
- Delete tasks
- Persistent storage using JSON

## Technologies
- Python 3
- JSON for local data storage
- Visual Studio Code

## Project Structure
- main.py → application logic
- tasks.json → data storage file (auto-generated)

## How It Works
The application runs in a loop, allowing users to interact with a menu to manage their tasks. All data is stored locally in a JSON file, ensuring tasks are saved between sessions.

## Skills Demonstrated
- Python fundamentals
- File handling (read/write JSON)
- Error handling
- Data structures (lists and dictionaries)
- CLI application design
